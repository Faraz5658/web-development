University Course Information

Introduction to Computer Science

Description: This course provides an introduction to the fundamental concepts of computer science. Topics include algorithms, programming, and problem-solving techniques. Students will gain hands-on experience with coding and computational thinking.

Syllabus:

  1. Week 1: Introduction to Programming
  2. Week 2: Variables and Data Types
  3. Week 3: Control Structures
  4. Week 4: Functions and Modules
  5. Week 5: Data Structures
  6. Week 6: Algorithms
  7. Week 7: Object-Oriented Programming
  8. Week 8: Final Project
